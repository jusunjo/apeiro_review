# 29cm Review Crawler

29cm 사이트의 상품 검색 결과를 기반으로 리뷰를 크롤링하고 Excel 파일로 저장하는 React 애플리케이션입니다.

## 기능

- 플랫폼 선택 (현재 29cm 지원)
- 검색어 입력
- 페이지 수 선택 (한 페이지당 50개 상품)
- 자동 리뷰 수집 (각 상품의 모든 리뷰)
- Excel 파일로 저장 (각 상품별로 열 생성)

## 설치

```bash
npm install
```

## 실행

```bash
npm run dev
```

## 빌드

```bash
npm run build
```

## 사용 방법

1. 플랫폼을 선택합니다 (현재는 29cm만 지원)
2. 검색할 키워드를 입력합니다
3. 수집할 페이지 수를 선택합니다 (한 페이지당 50개 상품)
4. "리뷰 수집 시작" 버튼을 클릭합니다
5. 수집이 완료되면 Excel 파일이 자동으로 다운로드됩니다

## Excel 파일 형식

각 상품별로 다음 컬럼들이 생성됩니다:
- 상품{itemNo}_옵션값
- 상품{itemNo}_사용자ID
- 상품{itemNo}_내용
- 상품{itemNo}_평점
- 상품{itemNo}_작성일시

## 주의사항

- API 호출 시 적절한 딜레이(500ms)를 두어 서버 부하를 방지합니다
- 상품 수와 리뷰 수에 따라 수집 시간이 달라질 수 있습니다
- 네트워크 오류나 서버 응답 지연 시 일부 리뷰가 누락될 수 있습니다
